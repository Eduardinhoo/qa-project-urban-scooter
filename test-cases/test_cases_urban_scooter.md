CT-001 – Cadastro com dados válidos

Objetivo: Garantir que o usuário consiga se cadastrar com dados corretos

Pré-condição: Usuário preecheu todos os campos com sucesso. 

Passos:

1 - Acessar a página da Urban Scooter
2 - Clicar no botão fazer pedido 
3 - Preencher todos os campos válidos
4 - Clicar no botão avançar 

Resultado esperado: Usuário preecheu todos os campos com sucesso e redirecionado para proxima pagina.


CT-002 - Cadastro da locação

Objetivo: Garantir que o usuário consiga se cadastrar com dados corretos

Passos:

1 - Preecher todos os campos validos 
2 - Marca as opções para validar o campo
3 - Clicar em fazer o pedido 
4 - Confirmar o pedido (sim) ou (Não)

Resultado esperado: Usuário preecheu todos os campos com sucesso vai exebir um mensagem com número do pedido, clicar no botão (verificar status) vai exibir todos os dados do pedido feito com sucesso.
