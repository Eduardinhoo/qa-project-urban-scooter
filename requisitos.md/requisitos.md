# Requisitos Funcionais – Urban Scooter

## RF-001 – Realizar pedido de scooter

**Descrição:**  
O sistema deve permitir que o usuário realize um pedido de scooter informando seus dados pessoais e endereço.

**Campos obrigatórios:**
- Nome
- Sobrenome
- Endereço de entrega
- Estação de metrô
- Telefone

**Regra:**  
Todos os campos devem estar preenchidos para avançar.

---

## RF-002 – Informar dados de locação

**Descrição:**  
O sistema deve permitir que o usuário informe os dados da locação da scooter.

**Campos obrigatórios:**
- Data de entrega
- Período de locação

**Campos opcionais:**
- Cor da scooter
- Comentário para o entregador

---

## RF-003 – Confirmar pedido

**Descrição:**  
Após finalizar o pedido, o sistema deve exibir:
- Número do pedido
- Opção de visualizar o status do pedido

**Resultado esperado:**  
Pedido realizado com sucesso e informações exibidas corretamente.
