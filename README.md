# Urban Scooter – Projeto de QA

Projeto de testes manuais desenvolvido com foco em simular o dia a dia de um Analista de QA Júnior.

## Objetivo
Garantir a qualidade do fluxo de pedido de scooters, validando regras de negócio, usabilidade e experiência do usuário.

## Escopo testado
- Pedido de scooter
- Dados do usuário
- Dados de locação
- Confirmação e status do pedido

## Técnicas utilizadas
- Testes funcionais
- Testes manuais
- Escrita de casos de teste
- BDD (Gherkin)
- Report de bugs

## Ferramentas
- GitHub
- VS Code
- Excel
- Gherkin (BDD)

## Observação
Projeto criado para fins educacionais e portfólio profissional.
